# BitQA的相关开发流程

克隆地址：

`git clone https://github.com/habaneraa/bit-qa-plus`

创建新的代码分支：

`git checkout -b <your_branch_name>`

修改、提交、推送三件套

`git add .`

`git commit -m "ADD: BitQA Git Develop Readme"`

`git push --set-upstream origin <your_branch_name>`
